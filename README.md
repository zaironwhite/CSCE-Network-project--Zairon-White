# CSCE-Network-project--Zairon-White

## Project Description

This project shows the design and realization of a little virtual client/server network in Cisco Packet Tracer.

## Network Equipment

- Cisco 1941 Router
- Cisco 2960 Switch
- Two Clients
- One Server

## Implementation Features

- IP Addresses Configuration
- DHCP 
- DNS 
- HTTP
- ICMP (Ping)
- Traceroute 

## Tests Done

The following tests were successfully done:

- Ping clients-server communication
- IP addresses configuration by DHCP
- DNS names resolution
- HTTP server access
- Traceroute

## Possible Improvements

- NAT
- HTTPS
- Wireshark packets monitoring
