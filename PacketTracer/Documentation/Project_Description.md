# Project Description

In this project, you will see a small client-server network that is built using Cisco Packet Tracer software.

## Devices Used
- Cisco 1941 Router
- Cisco 2960 Switch
- Two PCs
- One Server

## Network Functions
- Static IP addressing
- DNS
- HTTP
- ICMP (ping)
- TCP/IP protocol

## Tests Performed
- Ping test between PCs and server
- Name resolution
- HTTP site viewing
